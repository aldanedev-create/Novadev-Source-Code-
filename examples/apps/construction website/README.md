# Construction Website NovaDev Example

This example shows a larger NovaDev 1.0 project split across multiple `.nova`
files.

It uses:

- module imports
- separate table files
- separate page files
- separate route files
- custom frontend JavaScript
- custom frontend CSS
- backend Python wrapped inside NovaDev
- a generated custom Vue homepage

## Build

```bash
python nova.py build-fullstack "examples/apps/construction website"
```

Generated output:

```txt
generated/build-right-construction/
```
