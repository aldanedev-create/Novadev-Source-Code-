# Amazon Clone NovaDev Example

This example shows how NovaDev 1.0 can be used to create a custom e-commerce
project with high-level language declarations and custom website code.

The app is Amazon-inspired, but it uses original names, text, styling, and
starter data.

## Build

```bash
python nova.py build-fullstack "examples/apps/amazon clone"
```

Generated output:

```txt
generated/amazon-clone/
```

## What This Demonstrates

- `app`, `project`, `table`, `page`, and `route` declarations.
- Custom project architecture.
- A custom Vue storefront written from NovaDev.
- Imported custom JavaScript behavior.
- Imported custom CSS for the Amazon-style layout.
- Backend Python helper code and SQL customization.
