# NovaDev 0.4 Roadmap

Prototype 0.4 works as a learning language and app generator. It is still a toy
prototype, so these are natural next steps:

- Better syntax errors with source-line highlights.
- Real SQLite persistence for generated Flask apps.
- More frontend layouts and reusable components.
- Package/dependency declarations in NovaDev source.
- Importing NovaDev files from other NovaDev files.
- Tests written in NovaDev.
- More complete class behavior.

The current goal is clarity: a beginner can read the lexer, parser, AST, runtime,
and generator files and understand how a language becomes a working app.
