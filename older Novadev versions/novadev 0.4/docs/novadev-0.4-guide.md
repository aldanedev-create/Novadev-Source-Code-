# NovaDev 0.4 Guide

NovaDev 0.4 adds Python-like programming features and a full Flask project
builder.

Run code:

```bash
python nova.py run examples/python_features.nova
```

Use the shell:

```bash
python shell.py
```

Build the e-commerce app:

```bash
python nova.py build examples/alchicken_app.nova
```

Inside the shell, use the same command with a dot:

```txt
nova> .build examples/alchicken_app.nova
```

Read the detailed docs in `novadev/docs/`.
