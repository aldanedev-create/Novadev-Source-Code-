# NovaDev Package Roadmap

NovaDev 1.0 has a working lexer, parser, AST, interpreter, runtime, module
resolver, shell, CLI, and project generators.

Next practical improvements:

- Better line-specific semantic diagnostics.
- More complete plugin/package execution hooks.
- Optional type checking.
- Database migrations for generated backends.
- Test generation for frontend and backend projects.
- A stronger `nova.py dev` command that starts coordinated dev servers.
