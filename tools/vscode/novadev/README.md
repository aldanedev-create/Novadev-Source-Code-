# NovaDev VS Code Extension

This local extension registers `.nova` files as NovaDev source files, adds basic syntax highlighting, snippets, and ND icon metadata.

The NovaDev installer can copy this folder into:

```text
%USERPROFILE%\.vscode\extensions\novadev.novadev-language-1.1.0
```

Restart VS Code after installing NovaDev. If your current file icon theme does not use language icons, choose `NovaDev Icons` from `File Icon Theme`.
