# Roadmap

NovaDev 1.0 establishes the real language shape:

- Lexer, parser, AST, interpreter, runtime, module resolver, semantic analyzer,
  generators, shell, and CLI.
- General-purpose programming syntax for scripts and tools.
- Project declarations for full-stack applications.
- Vue and backend target generation.
- Safe Python bridge and generated custom code support.

Useful next improvements:

- Richer static analysis with line-specific diagnostics.
- More complete package/plugin installation.
- Stronger type hints and optional type checking.
- Database migrations for generated backends.
- More frontend component templates.
- Automated dev server orchestration from `nova.py dev`.
