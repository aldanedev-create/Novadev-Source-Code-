# Modes

Modes tell NovaDev what kind of project is being generated.

Supported common modes include:

```txt
custom, ecommerce, construction, crm, school, portfolio, restaurant, booking,
dashboard, blog, cms, church, gym, inventory, delivery, realestate,
healthcare, finance, trading, security, nonprofit, event, hotel, salon,
learning, marketplace, social, forum, projectmanagement, invoice, pos,
supportdesk, logistics
```

Domain modes provide validation hints and generation context. For example,
`mode ecommerce` expects products, carts, checkout, and orders. `mode
construction` expects services, projects, leads, and estimates.
